# aisence 채널 운영 사용방법

## 전체 프로세스 요약

```
1단계: 음원 기획 (Claude와 대화)
        ↓
2단계: 음원 생성 (Suno에서 직접 생성 후 수동 다운로드)
        ↓
3단계: 영상 제작 (자동 - merge_video.py)
        ↓
4단계: 유튜브 업로드 (자동 - 스크립트)
```

---

## 1단계 — 음원 기획

**유저가 할 것:**
- Claude에게 원하는 음악 느낌/콘셉트 말해주기
- 예시: "봄에 듣기 좋고 벚꽃이 휘날리는 느낌의 노래"

**Claude가 해주는 것:**
1. 분위기에 맞는 한국어 가사 작성 (10곡 이상)
2. 각 곡에 맞는 Suno 스타일 프롬프트 작성
3. 곡 목록 + 진행 상태 트래킹 표 제공
4. `docs/music_plan.md`에 기획 내용 기록

---

## 2단계 — 음원 생성

**유저가 할 것:**
1. Claude가 만들어준 가사 + 프롬프트를 Suno에 입력
2. 생성된 mp3 다운로드
3. `music_file/[시리즈명]/` 폴더에 저장
   - 파일명 규칙: `곡제목_1.mp3`, `곡제목_2.mp3` (2개 버전)
4. 더 마음에 드는 버전 선택

> ⚠️ Suno 공식 API 없음 → 웹에서 직접 생성 후 수동 다운로드

---

## 3단계 — 영상 제작

**유저가 할 것:**
1. 캔바에서 배경 이미지 제작 → 다운로드 (`images/` 폴더에 저장)
2. 아래 명령어 실행

**Claude가 해주는 것 (자동):**
- MP3 10개 이상을 순서대로 이어붙이기
- 이미지 1장을 배경으로 합성
- 1080p 또는 4K, 30fps, MP4 출력
- 첫 곡 페이드인 / 마지막 곡 페이드아웃 자동 적용

**실행 방법:**
```bash
# 폴더 통째로 넣기 (파일명 순 자동 정렬)
python scripts/merge_video.py \
  --image images/cover.png \
  --audio_dir music_file/[시리즈명] \
  --resolution 1080p

# 또는 파일 직접 나열 (순서 지정)
python scripts/merge_video.py \
  --image images/cover.png \
  --audio music_file/시리즈/1.mp3 music_file/시리즈/2.mp3 ...
```

**옵션:**
| 옵션 | 기본값 | 설명 |
|------|--------|------|
| `--resolution` | `1080p` | `1080p` 또는 `4k` |
| `--fps` | `30` | 프레임레이트 |
| `--fade_in` | `2.0` | 첫 곡 페이드인 (초) |
| `--fade_out` | `3.0` | 마지막 곡 페이드아웃 (초) |

**출력:** `output/merged_final.mp4`

---

## 4단계 — 유튜브 업로드

**유저가 할 것:**
- 완성된 mp4 파일 경로 Claude에게 알려주기
- 예시: "output/merged_final.mp4 올려줘"

**Claude가 해주는 것 (자동):**
- 영상 분위기에 맞는 **제목** 작성 (장르별 공식 적용)
- **설명** 작성 (제작 툴 Suno AI 명시 포함)
- **태그** 작성 (공통 태그 + 장르별 태그)
- YouTube Data API v3로 자동 업로드
- 업로드 예약 (기본: 금요일 오후 6시)

**업로드 내용 예시:**
```

설명:

오늘의 트랙: 카페 공부 플레이리스트
장르: Lo-fi, Chill, Study Music

──────────────────────────────
🎵 AI로 만든 감성 음악 채널, aisence
🛠️ 제작 툴: Suno AI
매주 새로운 트랙을 업로드합니다.

구독 & 좋아요는 큰 힘이 됩니다 🙏
──────────────────────────────

📌 이런 분께 추천해요
• 카페에서 공부할 때
• 집에서 작업 집중할 때
```
