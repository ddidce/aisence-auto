# aisence 채널 운영 사용방법

## 전체 프로세스 요약

```
1단계: 음원 기획 (Claude와 대화)
        ↓
2단계: 음원 생성 (자동 - Suno API)
        ↓
3단계: 영상 편집 (수동 - 캡컷)
        ↓
4단계: 유튜브 업로드 (자동 - 스크립트)
```

---

## 1단계 — 음원 기획

**유저가 할 것:**
- Claude에게 원하는 음악 느낌/콘셉트 말해주기
- 예시: "봄에 듣기 좋고 벚꽃이 휘날리는 느낌의 노래"

**Claude가 해주는 것:**
- 분위기에 맞는 한국어 가사 작성
- Suno 스타일 프롬프트 작성
- 40~50분 분량 커버하는 곡 목록 구성

---

## 2단계 — 음원 생성

**유저가 할 것:**
- 없음 (전부 자동)

**Claude가 해주는 것:**
- Suno API로 mp3 자동 생성
- `music_file/` 폴더에 자동 저장

**생성 결과:**
- 요청 1회당 2개 버전의 mp3 생성됨
- 더 마음에 드는 버전 선택해서 사용

---

## 3단계 — 영상 편집

**유저가 할 것:**
1. `music_file/` 폴더에서 mp3 파일 확인
2. 캡컷 열기
3. mp3 파일들을 순서대로 불러와서 하나로 합치기
4. 배경 이미지 (`music_file/AIsenseimg.png`) 적용
5. mp4로 내보내기 → `output/` 폴더에 저장

**Claude가 해주는 것:**
- 없음

---

## 4단계 — 유튜브 업로드

**유저가 할 것:**
- 완성된 mp4 파일 경로 Claude에게 알려주기
- 예시: "output/벚꽃시리즈_final.mp4 올려줘"

**Claude가 해주는 것:**
- 영상에 맞는 제목 / 설명 / 태그 작성
- 스크립트로 유튜브 자동 업로드
