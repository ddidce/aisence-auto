## 제목 공식

### 로파이 (LOFI)
```
[분위기 키워드] lofi hip hop ~ [상황] 🎵 aisence
```
**예시:**
- `새벽 감성 lofi hip hop ~ 공부할 때 듣기 좋은 음악 🎵 aisence`
- `빗소리 카페 lofi ~ 혼자만의 시간 🎵 aisence`
- `따뜻한 밤 lofi beats ~ 잠 못 이루는 새벽에 🎵 aisence`

### 힐링 (HEAL)
```
[감정/상황] 힐링 음악 ~ [효과] 🎵 aisence
```
**예시:**
- `지친 하루 끝에 듣는 힐링 음악 🎵 aisence`
- `마음이 편해지는 잔잔한 멜로디 🎵 aisence`
- `감성 힐링 BGM ~ 쉬고 싶을 때 🎵 aisence`

### 베이스 (BASS)
```
[분위기] bass music ~ [상황/감성] 🎵 aisence
```
**예시:**
- `딥 베이스 새벽 음악 ~ 혼자 드라이브할 때 🎵 aisence`
- `무거운 베이스 감성 ~ 집중력 높이는 음악 🎵 aisence`
- `dark bass vibes ~ 새벽 3시 🎵 aisence`

### 새벽감성 (DAWN)
```
새벽 [시간]시 감성 ~ [분위기 키워드] 🎵 aisence
```
**예시:**
- `새벽 3시 감성 ~ 아무도 없는 거리를 걸을 때 🎵 aisence`
- `잠 못 드는 새벽에 듣는 음악 🎵 aisence`
- `도시의 새벽 감성 BGM 🎵 aisence`

### 드라이브 (DRIV)
```
[시간대] 드라이브 음악 ~ [분위기] 🎵 aisence
```
**예시:**
- `새벽 드라이브 음악 ~ 아무 생각 없이 달릴 때 🎵 aisence`
- `야간 드라이브 감성 플레이리스트 🎵 aisence`
- `혼자 드라이브할 때 듣기 좋은 음악 🎵 aisence`
