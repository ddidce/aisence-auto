# AISENCE — 영상 편집 방향 가이드

---

## 기본 영상 구조

```
[페이드인 1~2초] → [메인 곡 전체] → [페이드아웃 2~3초]
```

단순하지만 감성적인 마무리가 핵심.

---

## 이미지 선택 기준

### 캔바에서 이미지 만들 때 원칙
1. **세로형 X, 가로형 (16:9) 유지** — 유튜브 기본 비율
2. **텍스트 최소화** — 채널명(`aisence`) + 분위기 단어 1개 정도
3. **장르별 무드 유지** (아래 컬러 가이드 참조)
4. **고해상도** — 최소 1920x1080 저장

### 장르별 이미지 무드

| 장르 | 추천 이미지 소재 | 색감 |
|------|---------------|------|
| 로파이 | 창문, 책상, 빗방울, 카페 내부, 고양이 | 따뜻한 오렌지/황색 |
| 힐링 | 자연 풍경, 꽃, 빛, 숲, 하늘 | 파스텔/밝은 톤 |
| 베이스 | 도시 야경, 지하도, 스모그, 빌딩 | 다크/퍼플/블루 |
| 새벽감성 | 새벽 거리, 가로등, 혼자 걷는 실루엣 | 딥블루/남색 |
| 드라이브 | 고속도로, 차 내부, 야간 도로, 터널 | 네온/다크 |

## Python 영상 자동 생성 설정

`moviepy` 기준 기본 설정값:

```
해상도: 1920 x 1080 (Full HD)
프레임레이트: 24fps
페이드인: 2초
페이드아웃: 3초
오디오 볼륨 정규화: -14 LUFS (유튜브 권장)
출력 포맷: .mp4 (H.264)
```

---

## 유튜브 업로드 설정

| 항목 | 설정값 |
|------|--------|
| 공개 설정 | 예약 업로드 (금요일 오후 6시) |
| 카테고리 | 음악 (Music) |
| 언어 | 한국어 |
| 자막 | 불필요 |
| 최종 화면 | 구독 버튼 + 다른 영상 추천 (마지막 20초) |
| 카드 | 관련 영상 링크 |

## 캡컷 사용 시 주의사항

캡컷으로 영상 내보낼 때:
- **해상도:** 1080p 이상
- **프레임:** 24fps 또는 30fps
- **포맷:** MP4
- **오디오 포함 여부:** 음악 포함해서 내보내기
  → Python에서 별도 오디오 합성 불필요

캡컷 없이 Python만 쓸 때:
- 이미지 파일 + mp3 파일만 있으면 자동 영상 생성 가능
- 가장 빠른 방법

---

## 파일 네이밍 규칙

```
[장르코드]-[날짜]-[버전번호]

예시:
  LOFI-20260315-001.mp3       ← Suno 원본 오디오
  LOFI-20260315-001.png       ← 캔바 이미지
  LOFI-20260315-001_final.mp4 ← 최종 업로드 영상
```
