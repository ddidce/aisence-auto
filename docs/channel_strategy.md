# AISENCE — 채널 운영 전략 & 업로드 스케줄

---

## 채널 정체성

| 항목 | 내용 |
|------|------|
| 채널명 | aisence |
| 콘셉트 | AI가 만드는 감성 음악 — 로파이, 힐링, 베이스, 새벽, 드라이브 |
| 업로드 주기 | 주 1회 |
| 권장 업로드 요일 | **금요일 오후 6시** (주말 전, 저녁 감성 타깃) |
| 영상 길이 | 2분 30초 ~ 4분 (단곡 기준) |

---

## 월간 장르 로테이션

4주 기준 장르를 순환하여 다양성 유지:

| 주차 | 장르 | 예시 키워드 |
|------|------|------------|
| 1주차 | 로파이 | 카페, 빗소리, 새벽 공부 |
| 2주차 | 새벽감성 | 새벽 3시, 혼자, 도시 |
| 3주차 | 드라이브 | 야간 드라이브, 고속도로, 자유 |
| 4주차 | 힐링 or 베이스 | 교차 업로드 |

> 특별 기획: 시즌별 테마 (봄 감성, 여름 새벽, 겨울 감성 등)

## 업로드 워크플로우 (주간)

```
월요일   — 이번 주 장르/분위기 결정, 나에게 프롬프트 요청
화요일   — Suno에서 음악 생성 & 다운로드 (3~5개 버전 중 선택)
수요일   — 캔바에서 이미지 제작
목요일   — 캡컷 or Python으로 영상 완성
금요일   — 유튜브 업로드 (오후 6시 예약 업로드)
토~일    — 댓글 반응 확인, 다음 주 아이디어 메모
```

## 썸네일 전략

**기본 원칙:**
- 텍스트 최소화 (채널명 + 핵심 단어 1~2개만)
- 어두운 톤 or 감성적인 색감 (채널 무드 통일)
- 캔바 템플릿 고정 사용 → 채널 아이덴티티 형성

**썸네일 색감 가이드:**
| 장르 | 추천 색감 |
|------|----------|
| 로파이 | 따뜻한 오렌지/갈색 톤 |
| 힐링 | 파스텔 그린/블루 |
| 베이스 | 다크 퍼플/블랙 |
| 새벽감성 | 딥블루/네이비/보라 |
| 드라이브 | 네온 + 다크 배경 |

## 성장 전략

### 초기 (0~50 구독자)
- 일관된 업로드 주기 유지가 최우선
- 해시태그 적극 활용
- 같은 장르 채널 커뮤니티 참여

### 중기 (50~500 구독자)
- 인기 영상 장르 위주로 집중
- 영상 길이 실험 (단곡 vs 1시간 컴필레이션)
- 커뮤니티 탭 활용 (다음 곡 투표 등)

### 장기 (500+ 구독자)
- 1시간짜리 플레이리스트 영상 제작
- 시즌/테마 시리즈 기획
- 쇼츠(Shorts) 병행 운영 (15~30초 예고편)

---

## 유튜브 쇼츠 활용 (선택)

메인 영상의 가장 좋은 30초를 잘라서 쇼츠로도 업로드:
- 추가 노출 확보
- 메인 채널로 유입 유도
- 캡션: `"풀버전은 채널에서 👆"`
